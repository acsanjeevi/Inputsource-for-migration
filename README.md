# Inputsource for Migration

## Cypress + TypeScript UI Automation (ICEPOT Framework)

### Project Structure
```
cypresswithtypescript/   — Full Cypress + TypeScript automation framework
```

### Target Application
SauceDemo: https://www.saucedemo.com

### Quick Start
```bash
cd cypresswithtypescript
npm install
npm run cy:run   # Run all tests headless
npm run cy:report  # Generate HTML report (timestamp + username)
```

See [ARCHITECTURE.md](cypresswithtypescript/ARCHITECTURE.md) for full framework documentation.
